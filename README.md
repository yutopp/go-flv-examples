# go-flv-example

Example programs which using [go-flv](https://github.com/yutopp/go-flv) library.

## How to build

```
make
```

Programs are generated under `dist` directory.

## Programs

### cmd/dump

`dump` outputs flv header data and tag data.

#### Usage

```
Usage of dist/dump:
  -i string
    	Input file

```
